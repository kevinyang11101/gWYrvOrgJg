# 项目简介

## 核心功能

- 智能学习计划生成 - 基于用户目标和上传教材，AI 自动生成个性化学习计划
- 智能组卷系统 - 支持自定义难度和题型分布，AI 自动生成试卷
- 知识图谱构建 - 自动提取知识点并构建知识图谱，支持交互式探索
- AI 问答功能 - 多模型支持，具备上下文记忆和教师人格切换能力
- 学习数据分析 - 提供测验成绩统计、错题分析和学习进度跟踪
### 用户功能

1. **注册/登录** - 创建账户并登录系统
2. **上传教材** - 上传 PDF/DOCX/PPTX 格式的学习材料
3. **生成学习计划** - 基于教材和目标生成个性化学习计划
4. **智能组卷** - 配置试卷参数，AI 自动生成试卷
5. **知识图谱** - 查看知识点关系图谱
6. **AI 问答** - 与 AI 助手对话，获取学习帮助

### 管理员功能

1. **模型管理** - 配置和管理 AI 模型
2. **Prompt 管理** - 编辑和管理 Prompt 模板
3. **API 日志** - 查看 API 调用记录和统计
4. **用户管理** - 管理用户账户
## 适用场景

- 个性化学习指导：为学生提供定制化的学习路径
- 知识体系构建：帮助用户系统化梳理和掌握知识点
- 自主学习辅助：支持用户通过问答交互获取知识解答
- 学习效果评估：通过数据分析帮助用户了解学习进度和薄弱环节
- 学术研究支持：基于论文库提供专业领域知识解答

## 技术特点

- 基于 Python 开发
- 集成 AI 大模型技术
- 结合知识图谱技术
- 依托 10,000+ 论文数据支持
